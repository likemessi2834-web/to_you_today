# 기능 명세서 (Features)

## 프로젝트 개요

**오늘의 당신에게 (To You Today)**는 시니어 사용자를 위한 Pinterest 스타일의 명언 카드 제작 및 공유 앱입니다.

## 제품 철학

- **시니어 중심 UX**: 직관적이고 단순한 인터페이스
- **Pinterest-like 피드**: 시각적으로 매력적인 Masonry 레이아웃
- **만들기와 공유**: 사용자가 쉽게 카드를 만들고 공유할 수 있음

## 현재 구현된 기능

### 홈 피드
- ✅ Masonry 그리드 레이아웃 (2열)
- ✅ 카테고리별 필터링 (칩 UI)
- ✅ 검색 기능 (명언 텍스트 및 카테고리 검색)
- ✅ 카드 공유 버튼

### 카드 제작
- ✅ 내 사진으로 꾸미기 (갤러리에서 선택)
- ✅ 추천 사진으로 꾸미기 (랜덤 이미지)
- ✅ 텍스트 오버레이 편집
  - 글씨체 선택 (10가지 한글 폰트)
  - 글자 크기 조절
  - 글자 색상 변경
  - 글자 배경 설정
  - 텍스트 정렬 (왼쪽/가운데/오른쪽/양쪽)
  - 텍스트 위치 드래그 이동
- ✅ 문구 추천 기능
- ✅ 직접 문구 입력 기능

### 저장 및 공유
- ✅ 스크린샷 캡처
- ✅ 갤러리 저장 (Android, saver_gallery)
- ✅ 공유 기능 (share_plus)
- ✅ "저장 vs 공유" 분리 UI

## 기술 스택

### 현재 사용 중
- Flutter 3.2.0+
- StatefulWidget 기반 상태 관리
- `cached_network_image`: 네트워크 이미지 로딩
- `flutter_staggered_grid_view`: Masonry 레이아웃
- `image_picker`: 갤러리 이미지 선택
- `screenshot`: 카드 캡처
- `share_plus`: 공유 기능
- `saver_gallery`: 갤러리 저장
- `google_fonts`: 한글 폰트

### 향후 검토 사항
- 상태 관리: 필요 시 Riverpod 도입 검토
- 백엔드: 향후 단계에서 결정

## 사용자 시나리오

1. **카드 탐색**
   - 홈 화면에서 Masonry 피드 탐색
   - 카테고리 칩으로 필터링
   - 검색으로 원하는 명언 찾기

2. **카드 제작**
   - "+" 버튼 클릭
   - "내 사진으로 꾸미기" 또는 "추천 사진으로 꾸미기" 선택
   - 텍스트 편집 (폰트, 색상, 크기, 위치 등)
   - "저장 및 공유" 버튼 클릭

3. **카드 공유**
   - 완성된 카드 미리보기
   - "저장(갤러리)" 또는 "공유하기" 선택
   - 갤러리에 저장하거나 다른 앱으로 공유

## 디자인 원칙

- **시니어 친화적**: 큰 터치 타겟, 명확한 레이블
- **시각적 계층**: 중요한 액션 강조 (노란색 버튼)
- **한글 폰트**: 다양한 한글 폰트로 감성 표현
- **깔끔한 UI**: 불필요한 요소 최소화
